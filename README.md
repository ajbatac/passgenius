# PassGenius

An advanced, local-first password generation suite (v0.3.0) featuring AI-driven optimizations, a premium cyber-futuristic interface, and a persistent bookmarklet for instant sidebar access.

## Key Features
- **Local-First Security:** No data ever leaves your device.
- **AI-Powered Suggestions:** Optimal configurations for banking, social media, and more.
- **Bookmarklet Sidebar:** Access PassGenius from any website window.
- **Premium UI:** Smooth animations and high-contrast dark mode.
