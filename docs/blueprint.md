# **App Name**: PassGenius

## Core Features:

- Configurable Password Generation: Generate passwords based on user-selected criteria, including length, inclusion of uppercase letters, lowercase letters, numbers, and symbols.
- Real-time Password Display: Instantly display the generated password, updating in real-time as selection options are changed.
- Live Password Strength Indicator: Provide immediate visual feedback on the strength of the generated password as options are adjusted.
- Exclude Ambiguous Characters: Option to prevent the generator from using easily confused characters like 'l', '1', 'I', 'O', '0'.
- One-Click Copy: Enable users to effortlessly copy the generated password to their clipboard with a single click.
- Smart Password Suggestion Tool: An AI-powered tool that suggests optimal password configurations (e.g., length, character sets) based on common security recommendations for different online service types.

## Style Guidelines:

- A sophisticated dark theme to convey security and allow vibrant colors to stand out. Primary elements, like interactive buttons and active states, will use a deep, rich indigo (#532AEE). The background will be a desaturated dark gray-blue (#201E27), creating a calm and secure environment. An accent color of bright, clean cyan-blue (#58CAF6) will be used sparingly for key highlights and notifications to add vibrancy.
- For headlines and numerical displays, 'Space Grotesk' (sans-serif) will be used for its clear, modern, and techy feel, accommodating large font sizes as requested. Body text and helper information will use 'Inter' (sans-serif) for high readability and versatility.
- Minimalist and clear line-style icons for actions like 'copy', 'refresh', 'settings', and strength indicators. Icons will be easily discernible and consistent with a modern, secure aesthetic.
- A clean, single-page, and intuitive layout that prioritizes the password generator controls and the generated output. Ample whitespace will ensure readability, especially for the prominent display of the generated password, with responsive design for various screen sizes.
- Subtle, fast animations will be used for real-time feedback: a quick glow or ripple effect upon password regeneration, a discreet checkmark animation when text is copied, and smooth transitions for the strength indicator updates.
